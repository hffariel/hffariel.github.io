# Cancel Label

Description NAME: cancel_label: cancel a transaction with label

SYNOPSIS curl -u user:passwd -XPOST http://host:port/api/{db}/{label}/_cancel

DESCRIPTION This command is used to cancel a transaction corresponding to a specified Label, which can be successfully cancelled during the Prepare phase.
