# lower

# Description

# Syntax

'INT lower (WARCHAR str)'

Convert all strings in parameters to lowercase

# example

mysql> SELECT lower("AbC123");
+-----------------+
| lower('AbC123') |
+-----------------+
| abc123          |
+-----------------+
